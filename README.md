# Kvasir
Kvasir (Vas-eer) is a support dashboard meant for WePay partners to be able to provide their end-users with basic support.  
It will perform:

    - account lookups

        - account statuses

        - withdrawal info

    - user lookups

    - resending user confirmation

    - refunding checkouts

    - account memebership management
